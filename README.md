IMPORTANT: For the sim to work you need to change the SDK.

Go to File->Project Structure->SDK:_______ 

Click Add SDK->Download JDK ->  Version 17, Bellsoft Liberica JDK (Full) -> Apply + OK

Check if the sim works. If something errors, you probably did something wrong.

To run the sim, simply click the green play button as if you were pushing code to the robot.
